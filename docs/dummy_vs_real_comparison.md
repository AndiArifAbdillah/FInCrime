## 🎯 Executive Summary (1 Paragraph)

FinCrime adalah sistem AML/CFT yang menggabungkan **algoritma production-ready** dengan **data 3-lapis**: (1) Algoritma & infrastruktur — 100% REAL (XGBoost, GraphSAGE, FastAPI, dll), (2) Data eksternal publik — 100% REAL (OFAC, UN, news, blockchain), (3) Data internal nasabah — DUMMY untuk demo (akan diganti via partnership). Project sudah verified end-to-end di sample sintetis ROC AUC 0.79, target 0.85+ saat retrain di data REAL (Elliptic Bitcoin Dataset 200k labeled).
